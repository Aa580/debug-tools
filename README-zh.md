# DebugTools

- 热重载，无需重启应用即可让编写的代码生效。
- 快速调用Java方法(本地/远程)。
- 搜索 http 地址以直接跳转到相应的方法定义。
- 打印SQL语句与耗时。
- 执行Groovy脚本。
